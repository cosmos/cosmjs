Interface PageRequest

PageRequest is to be embedded in gRPC request messages for efficient pagination. Ex:

message SomeRequest { Foo some_parameter = 1; PageRequest pagination = 2; }

countTotal

countTotal: boolean
  • Defined in cosmwasm-stargate/src/codec/cosmos/base/query/v1beta1/pagination.ts:40

count_total is set to true to indicate that the result set should include a count of the total number of items available for pagination in UIs. count_total is only respected when offset is used. It is ignored when key is set.

key

key: Uint8Array
  • Defined in cosmwasm-stargate/src/codec/cosmos/base/query/v1beta1/pagination.ts:22

key is a value returned in PageResponse.next_key to begin querying the next page most efficiently. Only one of offset or key should be set.

limit

limit: Long
  • Defined in cosmwasm-stargate/src/codec/cosmos/base/query/v1beta1/pagination.ts:33

limit is the total number of results to be returned in the result page. If left empty it will default to a value to be set by each app.

offset

offset: Long
  • Defined in cosmwasm-stargate/src/codec/cosmos/base/query/v1beta1/pagination.ts:28

offset is a numeric offset that can be used when key is unavailable. It is less efficient than using key. Only one of offset or key should be set.
