[ADR-020](https://github.com/cosmos/cosmos-sdk/blob/66c5798cec/docs/architecture/adr-020-protobuf-transaction-encoding.md)
and
[The 3 levels of proto encoding](https://warta.it/blog/cosmos-sdk-protobuf-signing).
